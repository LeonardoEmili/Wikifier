Java is a  |German_style_board_game|German_Style_Board_Game|  designed by  |Wolfgang_Kramer|Wolfgang_Kramer|  and  |Michael_Kiesling|Michael_Kiesling|  and published in 2000 by  |Ravensburger|Ravensburger|  in  |German|German|  and by  |Rio_Grande_Games|Rio_Grande_Games|  in English. It is illustrated by  |Franz_Vohwinkel|Franz_Vohwinkel| .     Java won the  |Deutscher_Spiele_Preis|Deutscher_Spiele_Preis|  9th place in 2001 and the  |Games_Magazine|Games|  Best Advanced Strategy Game in 2002. It is the second game in the  |Mask_Trilogy|Mask_Trilogy| , following  Tikal  and followed by  Mexica .     The game provides the atmosphere of the  |island|Island|  of  |Java|Java|  on a  |hexagonal_board|Hex_Map| . Players build the island and score by setting up  |palace|Palace| |festivals|Festival|  at opportune moments. When players run out of hexagons to build the island, the game is over. A final scoring phase now takes place and a winner is declared.
